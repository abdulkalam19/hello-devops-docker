# Hello DevOps Docker

This project demonstrates a simple static website deployed using *Docker* and *Nginx*.

## Project Overview
- Created a simple HTML web page.
- Used *Docker* to containerize the application.
- Used *Nginx* as the web server inside the Docker container.
- Built and ran the container on port 8082.
- Stored project in GitHub for version control.

## Commands Used
- Build Docker image:
  ```bash
  docker build -t hello-devops .
